
To build the docs

- Edit file Doxyfile.linux and set the path to vbfilter.sh

- From the Launcher directory, do

    ./Docs/doxyvb/make.sh

- Point your browser at file:///path/to/Launcher/Docs/doxyvb/html/index.html

- cd to Launcher/Docs/doxyvb/latex and do

    pdflatex refman.tex

- Copy

    Launcher/Docs/doxyvb/latex/refman.pdf

  to

    Launcher/Docs/XL-Launcher.pdf
