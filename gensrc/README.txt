
gensrc
<http://www.gensrc.org>

gensrc is a Python application which takes XML descriptions of functions
defined in a library and generates source code for addins on supported
platforms including Microsoft Excel and OpenOffice.org Calc. gensrc is used to
generate source code for the ObjectHandler and QuantLibAddin applications.

gensrc is a subproject of the QuantLib open source C++ analytics library.
QuantLib is Non-Copylefted Free Software.
QuantLib is OSI Certified Open Source Software.
OSI Certified is a certification mark of the Open Source Initiative.
