
QuantLibXL
<http://www.quantlibxl.org>

QuantLibXL exports the functionality of the QuantLib C++ analytics library to
Microsoft Excel.  QuantLibXL is a subproject of QuantLibAddin
(www.quantlibaddin.org) which implements the same interface on a variety of
other platforms.

The QuantLibXL project comprises a compiled Addin (an XLL) exposing QuantLib
functionality to Microsoft Excel, plus example workbooks and end-user
documentation.

QuantLib is Non-Copylefted Free Software.
QuantLib is OSI Certified Open Source Software.
OSI Certified is a certification mark of the Open Source Initiative.
